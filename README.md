# Game Mechanics Portfolio
Welcome to my Game Mechanics Portfolio! 

This repository contains the development diary blog posts featured on my portfolio website. The site documents the design and development of my retro-inspired 3D game built using Unreal Engine 5 and Blueprints Visual Scripting. 

This project was completed as part of my first-year university Game Design course for the Game Mechanics module and highlights my progress, challenges, and reflections throughout the development process.
