Game Mechanics Portfolio

Welcome to My Game Mechanics Module Project Portfolio Site!

This website documents the design and development of my retro-inspired 3D game, built using Unreal Engine 5 and Blueprints Visual Scripting, as part of my first-year university Game Design course for the Game Mechanics module at Birmingham City University. It includes my completed development diary blog posts, highlighting my progress, challenges, and reflections throughout the project.

About the Dev Diary

The dev diary was originally structured as weekly blog posts from Week 1 through Week 12 as part of the module project requirements. This format helped me document my progress and reflect on challenges regularly. Please note that some work for certain weeks was completed or refined in later weeks during review and catch-up sessions. The weekly structure is maintained for clarity and consistency.
